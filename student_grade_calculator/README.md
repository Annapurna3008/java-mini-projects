# Java Conditional Statements 

# Student Grade Calculator

## Objective

This program accepts student marks as input and determines the corresponding grade using Java conditional statements.

## Concepts used

Conditional Statements:
We use conditional statements to make decisions based on conditions that if a condition gets true then what happens and if not what will happen.

For example:
If we take an example that we want to check where a person is eligible for the voting or not. In general, a person is eligible to vote if his age is above 18 or else he can't vote. So, in these kind of cases we can use conditional statement where we know a condition and we have chose from different options.

In conditional statements we have if-else, if-else-if, switch case.
1) If-else is used to check one condition.
2) If-else-if is used when we have more than one condition.
3) Switch case is used one variable needs to be compared with multiple values instead checking different logical conditions.
   Example:
        In a restuarant we are given to enter a number where each number represents a different dish.
        1-> Gobi Manchuria
        2-> Chapathi
        3-> Panner curry
        and so on....


## Project Flow
1. Read marks from the user.
2. Compare marks using if-else-if statements.
3. Print the corresponding grade.
