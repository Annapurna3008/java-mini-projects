#Java varibales project

#Objective

This project demonstrates how to use variables and datatypes in Java.

#Concepts Used

Variables

Variables are container used to store the value. It helps in giving a name to the memory location.

Data Types

Data Types are 2 types:
1) Primitive Data types:
    There are 8 primitive data types:
    1) byte 
    2) short
    3) char
    4) boolean
    5) int
    6) long
    7) float
    8) double
2) Non-Primitive Data types:
    There are created by the programmers:
    1) String
    2) Array
    3) Class
    4) Object
    5) Interface

#About code
The program works like a calculator and used for simpler operations like addition, subtraction, multiplication, division and modulus.

In the program switch case is used. Based on the number entered by the user switch case perform thh operation.
For example:
    If user enter 1 then addition is performed 
    If user enter 2 then subtraction is performed and like-wise.
